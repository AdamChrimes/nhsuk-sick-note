Get an isolation note

Use this service if:

  • you have already been self-isolating for 7 days and need a note for your employer that you need to stay at home for longer
  • you have agreed with your employer that you cannot work from home
Information:

You can stay off work for up to 7 days without having to get a fit note or isolation note. Instead your employer can ask you to confirm that you've been ill. You can do this by filling in a form yourself when you return to work. This is called self-certification.

You can request an isolation note on behalf of someone else.

Start now

Before you start

If you are not sure if you need to stay at home, get advice now from the 111 coronavirus service.
