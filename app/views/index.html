Get a coronavirus self-isolation note

Use this service to get a note to confirm you have been told to self-isolate because of coronavirus.

Information:

You only need to self-isolate if advised to by NHS 111 or a medical professional.

We need to ask you a few questions to create the form. This takes around 2 minutes.

Start now

Before you start

This service is for people who have been told to self-isolate by the NHS 111 telephone service, the 111 coronoavirus service or a medical professional.

If you are not sure if you need to self-isolate, get advice now from the [111 coronavirus service] (https://111.nhs.uk/covid-19).
